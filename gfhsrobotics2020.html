Software Downloads - VEXCode is recommended.

VEXCode

This is the standard in coding software at the moment. I strongly adivse that you use it. All code from past years is written in and is compatible with. THIS IS THE ONLY SOFTWARE YOU CAN USE WITH WIRELESS UPLOADING!!!

VEX Coding Studio

Only use VEX Coding studio when viewing legacy code. You should not use to to write new code. It is a badly made program which means you will have a badly programmed robot.

Robot Mesh Studio
PROS

These last two are alternative programming methods. Neither have been used before and have no existing code written for this school. PROS is a more advanced piece of software and is similar to many IDEs which are common today. Robot Mesh Studio is a web based program that requires a plugin.
